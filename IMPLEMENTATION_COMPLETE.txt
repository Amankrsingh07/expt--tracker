🔐 SECURITY & SAFETY

   • User must be logged in to access budgets
   • Each user only sees own budgets
   • API enforces user ID checks

✅ Data Validation
   • Amount must be positive
   • Category must exist

   • Input sanitization

   • Database constraints prevent invalid entries
   • Clear error messages
